Ejercicios de Posicionamiento Grid 1

Ejercicio 6

En esta estructura realizada con el layout grid se trata de combinar grupos de celdas.

En el código HTML las celdas con el texto celda xx son los elementos escritos como bloques div.

La alineación del texto al centro es un estilo CSS com el color o el fondo.

La linea entre celdas es un gap de 1px. Intenta lograr el mismo aspecto que la figura.
